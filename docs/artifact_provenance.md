## Teacher boundary

The V173 teacher is not a manual label file. It is generated by an external
curriculum script that converts clean external table-tennis/badminton resources
into coarse priors such as action family, phase, landing depth, and response
patterns. Those priors are blended with AICUP fold-safe/task priors and existing
safe teacher targets. External datasets are not mapped directly to exact AICUP
`actionId` labels.

## What is reproducible from a fresh clone

1. The final submission artifact can be verified and copied exactly.
2. The component scripts and wrappers can be rerun after placing official data
   under `data/raw/` and optional external resources under `external_data/`.
3. The release does not redistribute official competition data, external raw
   datasets, or all historical cache/checkpoint files from hundreds of failed
   experiments.
